AD-Brokers is a Dynamic website that is about Billboards rental for companies and customers.
The website connects two parts; Marketing agencies that show their Bilboards and customers who need to rent billboard.

Technologies used: -
	1) HTML, CSS.
	2) Bootstrap 4.
	3) JavaScript (For form Validations).
	4) PHP and MySQL for Fetching and editing database.

To run the program you need xampp running on your device.
